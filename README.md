# Tanstack Table Test App

React table application with:
- filters, sorting,
- filter virtual lists,
- row selection,
- adding, deleting row,
- form validation,
- refreshing by websocket,
- bem methodology,
- FSD.

## Technologies

- Tanstack Table,
- Matherial UI,
- Tanstack Query,
- Axios,
- react-hook-form,
- yup,
- react-window,
- socket.io.
